## Purpose

Spread Chinese traditional colors to the world.

You can pick the color on [Coloring China](https://se.joway.io/).

## Usage

### Css

```html
<link rel="stylesheet" href="/lib/color-cn.min.css" />

// ...

<div class="anyuzi"></div>
<div class="bg-anyuzi"></div>
```

### Sass

`/lib/color-cn.scss`:

```scss
.mycls {
  color: $zhuganzi;
}
```
